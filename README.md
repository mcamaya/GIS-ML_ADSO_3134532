# GIS-ML  
### Gestor Inteligente de Stock para MercadoLibre

## 📌 Descripción del proyecto

GIS-ML es un sistema web desarrollado con el objetivo de automatizar la sincronización del inventario entre un sistema interno y la plataforma de comercio electrónico MercadoLibre.

Este proyecto surge como solución a un problema común en negocios digitales: la actualización manual del inventario, la cual genera inconsistencias, ventas de productos sin stock y errores operativos.

GIS-ML permite centralizar la gestión de productos y garantizar que el stock publicado en MercadoLibre refleje el inventario real del negocio en tiempo real o de forma automatizada.


## 🎯 Objetivo

Automatizar la actualización del inventario en MercadoLibre mediante integración con su API, reduciendo errores humanos y mejorando la eficiencia operativa del negocio.


## ⚙️ Funcionalidades principales

- 🔄 Sincronización automática de inventario
- 📦 Gestión de productos
- 🧩 Gestión de kits (productos agrupados)
- 🔐 Autenticación de usuarios
- 📊 Visualización de estado mediante dashboard
- 📝 Registro de logs y actividad del sistema
- 🔗 Integración con API de MercadoLibre


## 🏗️ Arquitectura del sistema

El sistema está basado en una arquitectura cliente-servidor:

- **Frontend:** React, HTML, CSS  
- **Backend:** Node.js, Express  
- **Base de datos:** MySQL  
- **Integración externa:** API de MercadoLibre  
- **Control de versiones:** Git
